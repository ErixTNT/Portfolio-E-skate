Proyecto E-Skate: Encuesta de Viabilidad

  


  

    
¿De qué trata este proyecto?

    

      Este trabajo tiene como objetivo presentar los resultados de una encuesta
      realizada para evaluar la viabilidad de un sistema de seguimiento y control
      de vehículos (especialmente skates eléctricos), como parte del proyecto
      E-Skate. El formulario fue compartido a un grupo reducido de personas para
      obtener un primer sondeo.
